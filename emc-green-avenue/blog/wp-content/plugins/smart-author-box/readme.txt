== Description ==

Smart Author Box will show the author's biography below articles. This is a responsive and clean designed author box at the end of your posts, which will display the author name, author gravatar and author description. It also allow to show the author social icon which values can be input from the wordpress user profile screen.

  
== Installation ==

1. Go to Plugins > Add New.
1. Search for "Smart Author Box".
1. Click "Install Now".

OR

1. Download the zip file.
1. Upload the zip file via Plugins > Add New > Upload.

Activate the plugin. Look for "Smart Author Box" in the admin menu.
